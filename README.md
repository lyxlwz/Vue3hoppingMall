## 创建项目
### 使用Vite脚手架 创建项目以及安装 路由和 状态管理
#### Vite脚手架 创建项目
```shell
# npm 6.x
$ npm init vite@latest <project-name> --template vue

# npm 7+，需要加上额外的双短横线
$ npm init vite@latest <project-name> -- --template vue
```
#### 安装路由
```shell
npm install vue-router@4
```
#### 安装状态管理
```shell
npm install vuex@next --save
```
